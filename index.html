# Data Deletion Request ## App Name: brain game This page explains how users can request deletion of their data. ### What data is collected * Nickname (user-entered) * Game progress and scores * Anonymous device identifier ### How to request deletion Users can request deletion of their data by sending an email to: 📧 [pocketaa@naver.com](mailto:pocketaa@naver.com) Please include: * Your nickname * Description of your request ### What happens after request * All associated data will be permanently deleted within 7 days ### Data retention * If no request is made, data may be retained for service operation * Some minimal logs may be retained for up to 30 days for security purposes ### Developer Information Developer: sunho park
