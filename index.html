Data Deletion Request

App Name: Memory Training: Brain Game

This page explains how users can request deletion of their data.

What data is collected

  • Nickname (user-entered)
  • Game progress and scores
  • Anonymous device identifier

How to request deletion

Email: pocketaa@naver.com

Please include:

  • Your nickname
  • Description of your request

What data will be deleted

  • Nickname
  • Game progress and scores
  • Associated anonymous identifiers

Data retention

Minimal logs may be retained for up to 30 days for security purposes.

Deletion timeline

All data will be deleted within 7 days of request.
